簡介 Link to heading

我是于京平（hicat0x0），目前就讀國立臺南大學資訊工程系（四年級），專注於資訊安全、無線通訊與系統工程。平時在校積極參與資安社與相關實作課程，透過比賽與專題累積實戰經驗，並持續學習逆向工程、漏洞分析與防禦技術。
